# food_delivery_application

This is an e-commerce app for food delivery using flutter with backend.
I also used firebase for notification.


Major features
1. Show products based on category
2. Shopping cart
3. Place your order
4. Track order
5. User profile
6. Sign in & Sign up
7. User address location from Google Map
8. Zone based order
9. User Authentication
10. Firebase notification
11. Payment (Paypal)
12. Create new products on the backend
13. See order details
14. Update order status
